Weather Application

A comprehensive weather application featuring user authentication with Firebase Auth and SharedPreferences for session management. Flight Delay Prediction
Python, TensorFlow, Scikit-Learn, Pandas, Matplotlib, Seaborn
A machine learning project that predicts flight delays based on multiple features including carrier, origin, destination, departure delay, and scheduled departure time. Implements comprehensive data preprocessing with categorical encoding and numerical feature normalization. Features correlation heatmap visualization and comparative model evaluation using both Random Forest classifier and neural network approaches for binary classification.
